# atividade

1. Execute o arquivo `algoritmo.py`:

```
python algoritmo.py
```

2. Selecione a opção desejada:

* `1` - Listar clientes
* `2` - Listar produtos
* `3` - Cadastrar cliente
* `4` - Cadastrar produto
* `5` - Excluir cliente
* `6` - Excluir produto
* `7` - Sair
